# Explain Code

Ask RaceEngineer to explain the selected code.

1. Select the code that you want to have explained in the editor.
1. Invoke the "Explain Code" command using one of the following options:
   1. Run the `RaceEngineer: Explain Code 💬` command from the command palette.
   1. Select the `RaceEngineer > Explain Code 💬` entry in the editor context menu (right-click).
1. The explanations shows up in the RaceEngineer sidebar panel.
