# Find Bugs

Identify potential defects in your code.

1. Select a piece of code that you want to check for bugs.
2. Invoke the "Find Bugs" command using one of the following options:
   1. Run the `RaceEngineer: Find Bugs 💬` command from the command palette.
   1. Select the `RaceEngineer > Find Bugs 💬` entry in the editor context menu (right-click).
3. RaceEngineer will show you a list of potential bugs in the chat window. You can refine it in the conversation panel.
